Instrucciones de levantamiento:
 - Primero en la carpeta raíz se debe usar el comando "npm install" a través de la terminal para instalar todas las dependencias.
 - Luego en la terminal en la carpeta raíz se debe usar el comando npm run dev.
 - Una vez ejecutándose hay que hacer ctrl+click sobre "http://localhost:5173/" para interactuar con la aplicación.
 - Luego en la carpeta raíz en un nuevo terminal paralelo al de la aplicacion se debe usar el comando "json-server --watch db.json --port 5000" para ejecutar el servidor ficticio json-server

Librerías que se usaron:
 - Axios
 - react-router-dom
 - Bootstrap
 - json-server
 - styled-components
 - react-icons
